# pld_comp
Projet de Compilation du groupe H4314

## Commande d'exécution
./pld_comp <Fichier.cpto> <options>

### Options
--token     : Affiche les tokens de l'arbre
--struct    : Affiche la structure de données
--ir        : Affiche l'IR
